SafeSkin AI

AI-Powered Skin Cancer Detection with Explainable Deep Learning

Project Overview

SafeSkin AI is a medical-grade ensemble deep learning system for early detection of melanoma and benign skin lesions. The system combines two CNN architectures to maximize recall while maintaining high precision.

Ensemble Architecture

Final Probability:
P = (P₁ + P₂) / 2

Where:
P₁ = DenseNet Output
P₂ = Inception Output

Optimized Threshold

Classification Rule:
Malignant if P > 0.173

Lower threshold chosen to reduce false negatives and increase sensitivity.

Explainable AI - GradCAM

GradCAM computes gradients of the predicted class with respect to convolutional feature maps:
αₖ = GlobalAveragePooling(∂y/∂Aₖ)

Heatmap = ReLU( Σ αₖ Aₖ )

This highlights clinically relevant lesion regions.
